## Notas sobre el canon

- **Los datos son de los databooks oficiales** (Official Character Book, Ultra Archive, Vol. 0, Guidebook) o de páginas oficiales (wiki MHA). Donde el canon no publica el dato, se marca como "No publicado". **No se inventan datos.**
- **El tipo de sangre y cumpleaños son datos fijos del canon.** El GM no los cambia (son parte de la identidad del personaje).
- **La comida favorita se puede usar narrativamente** — el PC puede cocinar para alguien, o el GM puede usar la comida como hook narrativo.
- **En la campaña 18+:** el tipo de sangre del PC (savia AB) es relevante para el sistema de drenaje y compatibilidad. Ver `docs/quirk_pc.md` sección "Drenaje de sangre".

## Distribución de tipos de sangre (resumen)

| Tipo | # personajes canónicos | Notas |
|---|---|---|
| **A** | ~24 | Más común en el canon (y en Japón real ~40%) |
| **B** | ~16 | Segundo más común |
| **O** | ~10 | Tercero, pero iconic por Midoriya |
| **AB** | ~6 | Menos común, raro |

Esto refleja la distribución real japonesa, con tipo A como el más común. El PC tiene tipo AB (rara), lo que hace su sangre valiosa e impredecible para el sistema de drenaje.

## Cómo se usa este doc en campaña

1. **Cumpleaños:** si el PC quiere celebrar el cumpleaños de un aliado, este doc le da la fecha.
2. **Cenas / cocina:** si el PC cocina para alguien, la comida favorita es un hook excelente.
3. **Citas:** un restaurante que sirva la comida favorita del interés es un bonus narrativo.
4. **Drenaje de sangre:** el tipo de sangre del NPC determina la compatibilidad con el PC (savia AB). Ver `docs/quirk_pc.md`.
5. **Recuperación por intimidad:** una cena romántica con la comida favorita del interés puede dar +10 a Closeness.
6. **Dramas familiares:** conocer la comida favorita de un villano (Toga) puede humanizarlo en una escena.
7. **NPCs no canon:** cuando se cree un NPC custom, se le asigna tipo de sangre, cumpleaños, y comida favorita. Se puede seguir la distribución del canon (mayoría A, luego B, etc.).
